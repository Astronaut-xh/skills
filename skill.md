# AI应用服务商落地计划 — 教培行业

## 你的画像
- **目标行业**：高中英语教培
- **产品定位**：师倍AI · 外刊英语周周练
- **核心差异化**：四大外刊题源 + AI改写 + 三层解析 + "我的收获"减负总结
- **时间**：在职，每天2-3小时（工作日晚上+周末）
- **AI能力**：熟练，能调API、用Coze/Dify、写脚本
- **客户资源**：从零开始

---

## 核心策略

> 先做"代工"赚第一笔钱 → 积累案例做"流程搭建" → 打磨成"行业方案"规模化

不选最完美的路，选最快收到钱的路。

---

## 产品形态：轻→重，三步走

> **不要做APP。** 开发成本高、获客门槛高、留存难。验证产品值不值得做 > 做一个漂亮的壳。

### 第一步：PDF/Word 周刊（0成本，现在就能干）

- **推什么**：每周一份PDF，发到老师微信里
- **内容**：1套阅读理解（4篇）+ 逐题解析 + 我的收获
- **交付**：微信直接发文件 / 邮件
- **为什么**：老师不需要装任何东西，打开就能用，甚至能直接打印发给学生
- **定价**：99元/月（每周1期，月付4期）

### 第二步：微信公众号（0成本，1天搭好）

- **推什么**：公众号每周推一期，付费阅读或社群订阅
  - 免费推文 = 本期试题（引流）
  - 付费部分 = 逐题解析 + 我的收获（1元/篇 或 年费会员）
  - 菜单栏 = 往期汇总 + 入群二维码
- **为什么**：微信是老师最高频工具，零学习成本；内容可搜索可转发，自带传播
- **进阶**：建微信社群，群内每天1道题互动，社群年费199元

### 第三步：小程序（低成本，验证后做）

- **什么时候做**：付费用户超过100人时
- **推什么**：微信小程序，学生/老师自助使用
  - 学生端：在线做题 → 即时出分 → 看解析 → 标记不会的
  - 老师端：选主题出卷 → 一键生成 → 导出PDF
  - 会员体系：免费试做3期 → 付费解锁
- **开发方式**：Coze应用内嵌小程序（1周）/ 外包极简版（2-5万）/ 微信云开发自写（0成本）

### 三步对比

| | PDF周刊 | 公众号 | 小程序 |
|---|---|---|---|
| 启动成本 | 0 | 0 | 2-5万或自己开发 |
| 搭建时间 | 今天 | 1天 | 1-4周 |
| 用户体验 | ★★★ | ★★★★ | ★★★★★ |
| 传播能力 | ★★ | ★★★★★ | ★★★★ |
| 变现能力 | ★★ | ★★★ | ★★★★★ |
| 适合阶段 | 验证期 | 起步期 | 规模期 |
| 用户门槛 | 零 | 零 | 需打开小程序 |

**核心原则：先卖PDF，再开公众号，最后做小程序。别倒过来。**

## Phase 1：准备期（第1-2周）

**目标：做出3个可展示的Demo，让自己"能卖"**

### 1.1 确定题型优先级（第1天）

高考英语题型多，不能一上来全做。按"分值高×外刊相关度高×AI可控"排序：

| 优先级 | 题型 | 推出时间 | 理由 |
|---|---|---|---|
| ★★★★★ | 阅读理解 | 第1个月 | 分值最高(30-35分)，外刊天然契合，AI质量可控 |
| ★★★★ | 完形填空 | 第2个月 | 外刊契合，AI难度高但差异化强，做壁垒 |
| ★★★ | 语法填空 | 第3个月 | 附赠，加套餐完整度，不单独推 |
| 暂不做 | 作文/听力/改错 | — | 听力需音频门槛高，作文改错差异化弱 |

**核心原则：阅读理解打天下，完形填空做壁垒，其他都是赠品。**

### 1.2 做出3个Demo（第2-14天）

3个Demo全部围绕英语，共用底层"外刊改写+解析"能力：

**Demo 1：AI外刊阅读理解出题系统**（核心 = 你的主产品）
```
输入：外刊文章URL/文本 + 难度（高考/模拟）
  ↓
AI筛选适配度 → AI改写（降词汇到3500词、控篇幅）→ AI出阅读理解题（4篇=A/B/C/D篇）
  ↓
输出：一套完整高考阅读理解 + 逐题解析 + 我的收获
```
- 工具：Coze/Dify工作流
- 展示形式：一份完整PDF样卷

**Demo 2：AI外刊完形填空出题系统**
```
输入：同一篇外刊文章，选择"完形填空"模式
  ↓
AI改写 → AI设计20个空（每空4选项，干扰项有逻辑）→ 生成解析
  ↓
输出：一篇完形填空 + 三层解析 + 我的收获
```
- 和Demo 1共享改写和解析模块，只换出题prompt
- 展示形式：一份完形填空PDF

**Demo 3：AI词汇精学卡片生成器**
```
输入：本期文章中的重点词汇（5-8个）
  ↓
AI生成：释义 + 搭配 + 熟词生义标注 + 1个例句 + 1个易混词
  ↓
输出：5-8张词汇卡片（可截图发学生/发小红书传播）
```
- 工具：Coze Bot
- 这是"我的收获"模块的独立产品化，也是内容传播素材

### 1.3 准备营销素材

- **一段话自我介绍**："我帮高中英语老师用AI从四大外刊自动出阅读理解题，附逐题解析和词汇精学，每周一套，不收钱先试"
- **1份PDF样卷**：完整的阅读理解一期（4篇+解析+我的收获）
- **3个Demo视频**：每个1-2分钟，展示从外刊到成卷的全流程
- **词汇卡片素材**：5-8张，用于小红书/抖音传播

## Phase 2：获客期（第3-6周）

**目标：拿到3-5个免费/低价客户，积累真实案例**

### 2.1 获客渠道（按优先级排列）

**渠道1：线下地推（最有效，但需要周末时间）**

- 周末去教培机构密集的区域（学校周边、写字楼）
- 目标：每周走访5-10家，聊上3-5家
- 话术模板：
  > "老师你好，我是做高中英语AI出题的。我们从纽约时报、经济学人这些外刊选文章，AI改编成高考难度，自动出阅读理解题，带逐题解析和词汇精学。你们平时出阅读理解大概花多少时间？我免费帮你们做一期试试。"

**渠道2：线上内容获客（每天1小时）**

- 小红书/抖音发AI英语出题演示视频
- 内容方向：
  - "AI从经济学人出一套高考阅读理解，老师看傻了"
  - "高考英语阅读理解的外刊题源，我帮你找好了"
  - "高中英语老师必备：1分钟出一套阅读理解"
- 每条视频末尾：有需要的老师私信我，免费体验
- 目标：每周发3-5条

**渠道3：教培行业社群**

- 加入教培行业微信/QQ群（搜索关键词：教培交流、培训机构、校长群）
- 不要直接发广告，先在群里免费分享AI工具使用技巧
- 建立信任后，私信聊合作

**渠道4：熟人介绍**

- 发朋友圈："最近在做高中英语AI出题，从外刊选文改写，自动出阅读理解+解析+词汇精学。有高中英语老师朋友可以推荐给我，免费帮做一期试试"
- 朋友的朋友就是最好的冷启动客户

## Phase 3：服务交付期（第7-12周）

**目标：稳定服务5-10个客户，月收入破万**

### 3.1 服务产品化

产品节奏与定价已在2.2节确定，此处聚焦交付标准化。

**每周交付物清单：**

| 交付项 | 内容 | 格式 |
|---|---|---|
| 阅读理解 | 4篇（A/B/C/D篇）+ 逐题解析 + 我的收获 | PDF |
| 完形填空（第2月起） | 1篇 + 逐题解析 + 我的收获 | PDF |
| 语法填空（第3月起） | 1篇 + 解析 | PDF |
| 词汇卡片 | 本期5-8个重点词汇 | 图片（可截图分享）|

### 3.2 交付流程标准化

把每天的工作固化成SOP：

**工作日（每天2-3小时）：**

```
19:00-19:30  处理客户需求（查看微信/消息）
19:30-20:30  执行AI代工任务（出题/批改/生成内容）
20:30-21:00  发1条小红书/抖音内容
21:00-21:30  维护工具/优化prompt/学习新工具
```

**周末（每天4-6小时）：**

```
上午  走访3-5家教培机构
下午  集中处理本周所有客户交付
晚上  做内容、复盘、优化流程
```

### 3.3 提效关键

你每天只有2-3小时，必须极致提效：

- **Prompt模板库**：把高频任务的prompt固化，一键调用
- **Coze/Dify工作流**：把重复操作自动化（输入→出题→排版→输出）
- **批量处理脚本**：一次生成10套卷子，而不是一套一套做
- **交付物模板**：固定的Word/PPT模板，AI产出直接填入

### 3.4 客户管理

- 用飞书/Notion做客户档案：需求、交付记录、续费情况
- 每个客户每周至少沟通1次，主动汇报效果
- 做到第2个月就问：有没有同行朋友也可以聊聊？

## Phase 4：规模化期（第13-24周）

**目标：月收入破3-5万，考虑是否全职**

### 4.1 从"代工"升级到"产品"

当你服务过10个客户后，会发现80%的需求是重复的。这时候：

1. 把重复需求做成**标准化AI应用**（Coze/Dify发布的应用）
2. 客户自助使用，你只做运维和答疑
3. 按账号收年费，而不是按月收服务费

**举例：**
- 搭一个"AI出题助手"应用，机构老师自己登录使用
- 定价：2999元/年/账号
- 你的成本：Coze Pro 99元/月 + 几小时维护

### 4.2 建立转介绍体系

- 每个老客户介绍新客户，双方各送1个月免费
- 做一份"效果报告"给客户：你这个月用AI省了XX小时，相当于省了XX元人力成本
- 客户拿这个报告给同行看，就是最好的销售材料

### 4.3 团队化（可选）

当客户超过15家，一个人可能忙不过来：
- 招1个兼职助手，处理基础代工任务
- 你专注在高价值环节：客户沟通、方案设计、产品迭代
- 兼职成本：2000-3000元/月

## 风险与应对

| 风险 | 应对 |
|---|---|
| 2-3小时不够用 | 严格按SOP执行，只做高价值动作；低价值动作用自动化 |
| 找不到客户 | 加大线上内容输出；调整话术，始终先免费试 |
| 客户觉得AI质量不够 | 先做辅助而非替代，AI出初稿+人工审核，降低预期 |
| 被客户白嫖 | 免费只做1次，明确"这是价值XX元的服务，本次免费体验" |
| 同行竞争 | 深耕一个细分领域，做到最懂这个行业 |
| AI工具成本上升 | 用开源模型替代；提前把prompt和工作流固化，不依赖单一平台 |
